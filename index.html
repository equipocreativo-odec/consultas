Información pedagógica
  • Marca una sola opción en esta tarjeta. Este recurso es un simulador con fines educativos.
  • El objetivo es que la ciudadanía practique la ubicación de casillas y la selección correcta.
  • Este simulador no es un sistema oficial de votación; no define resultados electorales.
  • No se capturan datos personales; solo se almacena la opción elegida y metadatos técnicos mínimos para el conteo pedagógico.
