
# Tarjetón Interactivo – una sola marca (simulación educativa)

Este tarjetón muestra **fotos** y **logos** por candidato y **solo permite una marca**, siguiendo la regla de voto válido (una sola elección).

## Estructura

```
index.html
assets/
  photo-<slug>.png  # Foto del candidato/a
  logo-<slug>.png   # Logo asociado
```

> Reemplaza cualquier imagen en `assets/` con el **mismo nombre de archivo** para usar fotos/logos reales.

## Cómo publicar
- Sube estos archivos a GitHub y activa **Pages** (Branch: `main`, Folder: `/`).
- Abre la URL pública que te dará GitHub Pages.

## Nota
Recurso educativo. No es un tarjetón oficial ni registra datos personales.
